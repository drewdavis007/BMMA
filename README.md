# Beautiful Minds Medical Aesthetics Website

Professional website for Beautiful Minds Medical Aesthetics - a premier medical aesthetics clinic in Brandon, Mississippi.

## Features

- Responsive design for all devices
- Modern pink and burgundy color scheme
- Services showcase (Botox, Dermal Fillers, Skin Care Products, Skin Pen)
- Contact form integration ready
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Node.js
- Express.js

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/beautiful-minds-website.git
cd beautiful-minds-website
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
node server.js
```

4. Open your browser and visit:
```
http://localhost:3000
```

## Adding Images

Place your images in the following locations:

- **Logo**: `public/images/logo.png`
- **Hero Background**: `public/images/hero-bg.jpg` or `hero-video.mp4`
- **About Section**: `public/images/about.jpg`
- **Service Icons**: `public/images/services/` folder

Then uncomment the image tags in `index.html` and remove placeholder text.
